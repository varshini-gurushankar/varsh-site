# varsh-site
personal website


**live at → [varshinigurushankar.com](https://varshinigurushankar.com/)**

## how i made it

- [Next.js 16](https://nextjs.org) (App Router) + React 19 + TypeScript
- [Tailwind CSS v4](https://tailwindcss.com) — theming via CSS variables, with a light/dark mode toggle
- [Framer Motion](https://motion.dev) — draggable floating windows, hover wiggles, and the theme-toggle sparkle burst
- [use-sound](https://github.com/joshwcomeau/use-sound) — click/close/bubble sound effects, with a mute toggle
- window state (open windows, z-order, theme, sound) lives in one React context
